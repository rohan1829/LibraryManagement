
# Library Management System (Java Console Project)

A simple console-based Library Management System built using Java and Object-Oriented Programming principles.

## Features
- Add new books to the library
- View all available books
- Borrow a book
- Return a book
- Uses OOP concepts like Classes, Objects, Encapsulation, and ArrayList

## Technologies Used
- Java
- OOP principles
- ArrayList for data storage
